# WebServer
A simple web server in C++ 98

## Organização do projeto

```
meu_projeto/
├── include/        # Arquivos .h (headers) públicos, visíveis para outras partes do projeto
│   └── minha_classe.hpp
│
├── src/            # Implementações (.cpp) do código
│   └── minha_classe.cpp
│
├── bin/            # Executáveis gerados
│   └── meu_programa
│
├── obj/            # Objetos compilados temporários (.o)
│   └── minha_classe.o
│
├── tests/          # (Opcional) testes unitários
│   └── teste_principal.cpp
|
├── config/          # Arquivo de configuração do WebServer
│   └── teste_principal.cpp
│
├── Makefile        # Script de build
└── README.md       # Documentação do projeto
```
