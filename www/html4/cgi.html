Demo CGI JSON DB

Use os botões para criar, ver e apagar itens. Tudo salvo em um arquivo chamado db.json.

1) Adicionar item (POST)

Cole aqui o objeto em JSON. Exemplo mínimo: {"id":1,"name":"Maria", "value": 10}

Pronto para enviar

2) Buscar (GET)

Clique em "Listar tudo" para ver todo o arquivo. Para buscar por id, digite o número e clique em "Buscar".

Resposta aparecerá no painel à direita

3) Remover (DELETE)

Coloque o id do item que quer apagar e pressione "Deletar".

Como testar (rápido)

Se preferir linha de comando use:

curl -i -X POST /cgi-bin/post.py -H "Content-Type: application/json" --data '{"id":2,"name":"Ana"}'
curl -i /cgi-bin/get.py
curl -i -X DELETE "/cgi-bin/delete.py?id=2"

Dica: o arquivo db.json fica na mesma pasta que os scripts Python. O servidor web precisa poder escrever nesse arquivo (permissões).
